## Vocation du projet 🚢

Ce projet a pour but de mettre en valeur les passagers du Titanic.
Si vous n'êtes pas connecté, vous y retrouverez simplement la liste des passagers et l’histoire du Titanic. 
En revanche, si vous trouvez l'adresse e-mail secrète et le mot de passe encore plus secret (c'est faux) vous pourrez alors avoir accès à de plus amples fonctionnalités. 
En effet,  il vous sera ainsi possible de trier les passagers par genre (♀️♂️) ou s’ils ont survécu ou non (🕊️❤️) à ce périlleux voyage.

## Lancer le projet 📋

Plusieurs étapes vous seront nécessaires pour ouvrir ce projet. Voici ce que vous allez devoir écrire dans la console :

1. Placez-vous dans le dossier `backend` et tapez la commande : `npm run dev`
2. Placez-vous dans le dossier `src` du dossier `front` et tapez la commande : `nmp run start`

Et voilà, le tour est joué, rendez-vous sur [http://localhost:4200](http://localhost:4200/) !

PS : pour le login, rien de plus simple admin@admin.fr avec le mdp admin suffira pour vous fournir le ticket d’entrée
